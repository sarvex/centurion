Detailed Description

A representation of an 8-bit RGBA color.

This class supports construction and conversion from various formats such as the SDL color types, HSV, HSL, normalized components, and hexadecimal RGB/RGBA/ARGB color codes.

A large collection of named color constants for commonly used HTML colors is provided in the cen::colors namespace.

This type is serializable, see the associated serialize function.
